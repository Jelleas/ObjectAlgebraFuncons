This directory contains our Caml Light (CL) test cases.  They are organised as follows:

  Advanced: Assorted larger programs testing combinations of features and corner-cases.
     Basic: Very small programs testing basic language features.
  Equality: Tests for structural and physical equality.
 Exception: Tests for throwing/catching exceptions, and evaluation order.
MuRecTypes: Tests for mutually recursive type definitions.
        OL: All the examples from Chapter 1 of the OCaml 4.0 User Manual
        PM: Tests for pattern matching.
 Shadowing: Tests for shadowing of identifiers (values, types and labels).
    Syntax: Tests for syntax extensions.
     Types: Testing type annotations and unusual type definitions.
    ValRes: Testing the value restriction.
